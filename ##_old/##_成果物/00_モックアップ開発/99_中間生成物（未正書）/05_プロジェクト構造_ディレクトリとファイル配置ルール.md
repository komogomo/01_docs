## 4. ファイル配置ルール

### 4.1 HTMLファイル

**配置場所:** `pages/[カテゴリ]/[画面名].html`

**例:**
```
pages/home/home.html
pages/board/board.html
pages/booking/booking.html
```

**命名規則:**
- 全て小文字
- 単語の区切りはハイフン（-）
- 拡張子は`.html`

### 4.2 CSSファイル

**共通CSS:** `css/common/[コンポーネント名].css`
- button.css
- header.css
- footer.css
- layout.css

**画面固有CSS:** `css/pages/[画面名].css`

**例:**
```
css/pages/home.css
css/pages/board.css
css/pages/booking.css
```

### 4.3 JavaScriptファイル

**共通機能:** `js/features/[機能名].js`
- language-switcher.js
- translator.js
- footer-navigation.js

**画面固有JS:** `js/pages/[画面名].js`

**例:**
```
js/pages/home.js
js/pages/board.js
js/pages/booking.js
```

### 4.4 翻訳データ

**配置場所:** `js/i18n/langs/[言語].js`

**ファイル:**
- ja.js（日本語）
- en.js（英語）
- zh.js（中国語）

**重要:** 全画面の翻訳データを統合して管理

## 5. 新画面追加手順

### Step 1: ディレクトリ作成

```bash
# 画面HTMLディレクトリ
mkdir pages/[画面名]/
```

### Step 2: テンプレートコピー

```bash
# テンプレートをコピー
cp templates/page-template.html pages/[画面名]/[画面名].html
```

### Step 3: CSS/JSファイル作成

```bash
# 画面固有CSS作成
touch css/pages/[画面名].css

# 画面固有JS作成
touch js/pages/[画面名].js
```

### Step 4: HTMLファイル編集

1. ファイルヘッダーコメント更新
   ```html
   <!--
     設置パス: /pages/[画面名]/[画面名].html
     画面名: [画面名]画面
   -->
   ```

2. CSS読み込みパス修正
   ```html
   <link rel="stylesheet" href="../../css/pages/[画面名].css">
   ```

3. JS読み込みパス修正
   ```html
   <script src="../../js/pages/[画面名].js"></script>
   ```

4. コンテンツ領域（`<main>`内）を編集

### Step 5: 翻訳キー追加

`js/i18n/langs/ja.js`, `en.js`, `zh.js`に翻訳キーを追加

```javascript
// ja.js
'[画面名].title': '[画面タイトル]',
'[画面名].content.message': '[メッセージ]',
```

## 6. 翻訳キー命名規則

### 6.1 基本構造

```
[画面ID].[カテゴリ].[項目]
```

### 6.2 カテゴリ

- **header** - ヘッダー関連
- **content** - コンテンツ領域
- **button** - ボタン
- **label** - ラベル
- **message** - メッセージ
- **error** - エラーメッセージ

### 6.3 例

```javascript
// ホーム画面
'home.header.title': 'ホーム',
'home.content.welcome': 'ようこそ',
'home.button.logout': 'ログアウト',

// 掲示板
'board.header.title': '掲示板',
'board.content.new': '新規投稿',
'board.button.post': '投稿する',

// 予約
'booking.header.title': '駐車場予約',
'booking.content.calendar': 'カレンダー',
'booking.button.reserve': '予約する',
```

## 7. 共通ルール

### 7.1 index.htmlの特例

- **配置:** プロジェクトルート直下
- **構造:** 独自構造（3層構造ではない）
- **CSS/JS:** インライン記述OK
- **変更:** 原則として変更不要

### 7.2 標準3層構造

全画面（index.html以外）は以下の3層構造に従う：

1. **ヘッダー領域**（`<header class="page-header">`）
   - 共通・編集不要
   - 言語切替ボタン
   - 画面タイトル

2. **コンテンツ領域**（`<main class="page-content">`）
   - 画面固有・編集可能
   - この部分のみカスタマイズ

3. **フッター領域**（`<footer class="page-footer">`）
   - 共通・編集不要
   - ナビゲーションボタン

### 7.3 ファイルヘッダーコメント（必須）

全てのHTML/CSS/JSファイルに以下を記載：

```html
<!--
  設置パス: /pages/[画面名]/[画面名].html
  画面名: [画面名]画面
-->
```

```css
/**
 * 設置パス: /css/pages/[画面名].css
 * ファイル名: [画面名]画面固有スタイル
 */
```

```javascript
/**
 * 設置パス: /js/pages/[画面名].js
 * 機能名: [画面名]画面固有処理
 */
```
