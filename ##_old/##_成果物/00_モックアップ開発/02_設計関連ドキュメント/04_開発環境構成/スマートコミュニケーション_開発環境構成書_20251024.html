1. プロジェクト概要

プロジェクト名: セキュレアシティ（SECUREA City）

概要: マンション管理組合向けスマートコミュニケーションアプリケーション

目的: 住民間のコミュニケーション円滑化、施設予約管理、掲示板機能の提供

開発体制: AI支援開発（Claude + Gemini連携）

7. 開発環境構築手順

7.1 前提条件

●  • Node.js 22.20.0 インストール済み

●  • Docker Desktop v28.5.1 インストール済み

●  • VSCode インストール済み

●  • Git インストール済み

7.2 リポジトリクローン

git clone https://github.com/komogomo/securea-city.git

cd securea-city

7.3 バックエンドセットアップ

cd backend

npm install

# .envファイルを作成し、環境変数を設定

npx prisma migrate dev

npm run start:dev

7.4 フロントエンドセットアップ

cd frontend

npm install

npm run dev

8. AI支援開発について

本プロジェクトはAI（Claude & Gemini）を活用した開発手法を採用しています。

8.1 使用AI

●  • Claude Sonnet 4.5: 設計・実装・デバッグ

●  • Gemini: セカンダリレビュー・コード検証

8.2 開発の特徴

●  • 要件定義から実装まで一貫したAIサポート

●  • コード品質の自動レビュー

●  • ベストプラクティスの自動適用

●  • 詳細なドキュメント自動生成
